# CreditWise Loan Approval Predictor

A Machine Learning web application that predicts whether a loan application is likely to be approved based on applicant details.

## Live Demo
https://creditwise-loan-approval-predictor-platform.streamlit.app/

## Features
- Loan approval prediction
- Random Forest Classifier
- Interactive Streamlit interface
- Confidence score for predictions

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Model Performance
- Algorithm: Random Forest Classifier
- Accuracy: **90%**
